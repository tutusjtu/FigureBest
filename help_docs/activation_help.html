重要注意事项

  • 首次激活/激活码激活
    • 必须联网选择 【方式二：激活码在线激活】
    • 根据如何获取激活码找到并输入图通道提供的激活码（形如：3f04432a94cea4576d04af7f0bcer429）
    • 成功后，自动生成与设备绑定的许可证（.lic 文件）
    • 许可证将自动添加到 MATLAB 搜索路径
    • 请确保网络畅通，并且安装FB的文件夹必须具有读取和写入权限（注意：MATLAB toolbox文件夹、OneDrive等网盘可能没有写入权限）！
  • 再次激活/许可证激活
    • 无需联网，选择 【方式一：许可证文件激活】
    • 定位到首次激活生成/图图发放的 .lic 文件
    • 成功后自动更新 MATLAB 搜索路径
  • 使用限制：
    • 激活次数 N = 设备数 n × 单设备可激活次数 m
    • 单设备重复激活将消耗额外次数
    • 具体次数以购买套餐为准
  • 文件管理：
    • 务必备份许可证文件；若购买激活次数为 N = 1，同一设备再次获得也需要再次付费！
    • 许可证绑定设备硬件，永久有效

如何获取激活码

您可以通过以下方式获取激活码：

  • 如果您还没有购买过FigureBest产品/想要为更多设备激活，请关注微信公众号“图通道”回复“FBV4”获取FigureBest/激活码
  • 如果您已购买过FigureBest产品/剩余激活次数充足，按下方指导查询激活码
  • 从 面包多 购买 FB —— 购买过 FB产品本身或任何包含 FB 的产品（如全家桶、小食桶），都可以按此方式获得激活码，所获得的激活码都可以用于激活FB！
  • 从 B站 购买 FB —— 购买过 FB 产品本身或任何包含 FB 的产品，都可以按此方式获得激活码，所获得的激活码都可以用于激活FB！

若在获取激活码过程中遇到问题（如：机密设备不能联网激活），请发送邮件至 865774179@qq.com，邮件主题请写明 【FigureBest激活码问题-[具体的订单号]】，并在邮件正文中附上(1)所有购买记录/付款记录/完整的报错截图;(2)运行代码`getDeviceSerial(54749110)`后获得的结果数据，否则不予回复。

分步操作指南

方式一：许可证文件激活

  1. 切换至 "许可证 | License File" 模式
  2. 点击输入框选择本地 .lic 文件
  3. 点击 "立即激活" 按钮
  4. 等待状态提示：
    • 绿色提示：激活成功

方式二：激活码在线激活

  1. 切换至 "激活码 | Activation Code" 模式：
  2. 输入图通道提供的激活码
  3. 点击激活按钮：
    • 显示 "正在连接服务器..."
  4. 激活成功后：
    • 显示打开许可证文件夹按钮，显示剩余激活次数
