重要注意事项

  • 首次激活/激活码激活
    • 必须联网选择 【方式二：激活码在线激活】
    • 根据 如何获取激活码 输入图通道提供的激活码（形如：3f04432a94cea4576d04af7f0bcer429）
    • 成功后生成设备绑定的许可证（.lic 文件）
    • 许可证将自动添加到 MATLAB 搜索路径
    • 请确保网络畅通，安装目录必须具有读写权限（MATLAB toolbox、OneDrive等路径可能没有权限）
  • 再次激活/许可证激活
    • 无需联网，选择 【方式一：许可证文件激活】
    • 定位至生成的 .lic 文件
    • 成功后自动更新 MATLAB 搜索路径
  • 使用限制：
    • 激活次数 N = 设备数 n × 每台设备可激活次数 m
    • 同一设备重复激活会额外消耗次数
    • 次数以购买套餐为准
  • 文件管理：
    • 请务必备份许可证；若 N = 1，重新获取需重新付费！
    • 许可证绑定设备硬件，永久有效

分步操作指南

方式一：许可证文件激活

  1. 切换至 "许可证 | License File" 模式
  2. 点击输入框，选择本地 .lic 文件
  3. 点击 "立即激活" 按钮
  4. 等待状态提示：
    • 绿色提示：激活成功

方式二：激活码在线激活

  1. 切换至 "激活码 | Activation Code" 模式
  2. 输入图通道提供的激活码
  3. 点击激活按钮：
    • 提示 “正在连接服务器...”
  4. 激活成功后：
    • 显示许可证文件夹按钮和剩余激活次数
